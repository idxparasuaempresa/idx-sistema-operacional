# IDX Sistema Operacional

Dashboard operacional completo para gestão de tráfego pago da IDX.

## Funcionalidades
- 📊 Dashboard com KPIs financeiros e métricas de campanhas
- 👥 Gestão de clientes (Meta Ads + Google Ads)
- 📋 Checklists diário, semanal e mensal com tarefas reais de gestão de tráfego
- 📓 Diário de otimizações e estratégias por cliente
- 📈 Histórico de métricas mensais
- 🔗 Guia passo a passo de conexão com Meta Ads
- 💡 Editor de estratégias no estilo Apple Notes
- 🎯 Dashboard com drag-and-drop e gráficos interativos

## Stack
- Frontend: HTML + CSS + Vanilla JS (single file, abre direto no navegador)
- Backend: Supabase (PostgreSQL + REST API)

## Setup do Banco de Dados
Execute o arquivo `migration_checklist_execucoes.sql` no **Supabase SQL Editor**:
1. Acesse https://supabase.com/dashboard/project/skkagtyesvtiujjszktu/sql
2. Clique em "New Query"
3. Cole o conteúdo do arquivo e execute

## Como Usar
Abra `index.html` direto no navegador — não precisa de servidor local.
